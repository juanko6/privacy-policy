Política de Privacidad para juanko.dev App

Última actualización: 17 de abril de 2026

Esta Política de Privacidad describe cómo Juan Carlos Gutiérrez Cáceres ("nosotros" o "el desarrollador") gestiona la información al utilizar la aplicación integrada con la API de LinkedIn.

1. Información que Recopilamos

A través de la API de LinkedIn, nuestra aplicación puede acceder a:

  • Información de perfil básico (nombre e identificador único de usuario).
  • Permisos para publicar contenido (w_member_social) en su nombre.

2. Uso de la Información

La información obtenida se utiliza exclusivamente para permitir la funcionalidad de programación y publicación de contenido en la red social LinkedIn por parte del usuario autorizado.

3. Almacenamiento y Seguridad

No compartimos sus datos con terceros. Los tokens de acceso se almacenan de forma segura y se utilizan únicamente para ejecutar las acciones solicitadas por el usuario.

4. Derechos del Usuario

Usted puede revocar el acceso de esta aplicación en cualquier momento desde la configuración de su cuenta de LinkedIn.
